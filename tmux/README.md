# Keybindings:

```
<C-b> - meta
<Meta-)> - next session
<Meta-(> - previous session
<Meta-s> - list/select sessions
<Meta-F> - highlight hashes/urls
<Meta-[> - visual mode
    * <space> - select text
    * <enter> - confirm selection, copies to clipboard
<Meta-]> - pase clipboard
```
